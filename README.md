# 🏅 Gestión de Eventos Deportivos y Recreativos

¡Bienvenidos al repositorio de la aplicación! Este proyecto consiste en un sitio web estático diseñado específicamente para la difusión, organización y visualización de eventos deportivos y actividades recreativas de carácter privado.

---

## 🚀 Objetivo del Proyecto

El propósito principal es desarrollar una aplicación web interactiva y atractiva centrada en la gestión de eventos deportivos y recreativos, asegurando un flujo de trabajo colaborativo y un control de versiones óptimo mediante el uso de **Git** y **GitHub**.

## 🛠️ Tecnologías Utilizadas

El sitio está construido utilizando las tecnologías esenciales del desarrollo web frontend:

| Tecnología | Descripción |
| :--- | :--- |
| **HTML5** | Estructuración semántica del contenido de los eventos. |
| **CSS3** | Diseño visual moderno, estilos atractivos y adaptabilidad a dispositivos móviles (Responsive Design). |
| **JavaScript** | Interactividad de la interfaz, manejo de filtros de eventos y dinamismo. |
| **Git & GitHub** | Herramientas utilizadas para el control de versiones y el trabajo en equipo. |
